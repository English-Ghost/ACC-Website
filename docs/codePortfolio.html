Chord Progression App

At the momemnt I am working on an app that allows users and easier way to find chord progressions in music and also show the chords that fit that porgression on the guitar. It is being made in Python and I am using TKinter as the GUI package for the base version of the program. So far in the code there are the lists of progressions one can have and they take you to another page in the program that will ask you for the key you want the chord progression to be in. It is still a WIP and future updates will happen. Being planned are the ways to incorporate showing the users the amount of different chords and variations on the fret board. The main hurdle at the momemnt is also learning the music theory in order for the app to work well.


Music Queue App

My friend and I are working on an app together that allows people at a gathering to be able to queue up songs depending on what music app the host is using. The way that it would work is that there is one host on the app/webiste that would link up whatever streaming platform they are using and others would connect through using a password. With the API from the given service we should be able to allow people to queue up songs to either a playlist or just the regular queue, either way they would be saved in a playlist in case the host wants to save the playlist from that night.
