Internships

So far I have had one internship during my freshman to sophmore year at BD Energy Systems. During this internship I was given a FORTRAN code and had the task of rewriting the FORTRAN code into Python code with a GUI so that the engineers would have easier access to the programs. The GUI was made using TKinter since nothing fancy was needed for the engineers to understand how the program worked. In total there were about 6 programs we recreated in this style and once we had gotten it almost finshed we would have an engineer test it. In order for that to happen we would let the engineer do the equations by hand and then once through the program and compare the answers. To much detail about the program can't be given since it was created to be sold. BD Energy Systems: http://www.bdenergysystems.com/
